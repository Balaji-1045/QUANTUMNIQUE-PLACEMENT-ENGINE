COMMUNICATION PRACTICE

Speak for 3 minutes on:

Explain how the Internet works

Include:
- What happens when you type a URL
- DNS
- Server
- Response



The Internet is a system that connects computers around the world
and helps them share information. When we type a URL in a browser,
the browser first tries to find where that website is located.
For this, it uses DNS, which works like a contact list and converts 
the website name into an IP address. After getting the IP address,
the browser sends a request to the server where the website is stored.
The server then processes the request and sends back the needed
data like web pages or images. Finally, the browser receives this
response and shows the website on the screen. This whole process
happens very quickly, so we can access websites in just a few seconds.
